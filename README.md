# mufengblog-shell

## USAGE

```bash
yarn global add mufengblog-shell

# deploy
mufengblog deploy -u <github userName> -p <vuepress docs dir> -r <github repository address>
# example
mufengblog deploy -u bloss -p docs -r zhangximufeng.github.io

# new post article
mufengblog post -p <file path> --page <filename>
# example
mufengblog post -p docs/hello --page index.md
```
